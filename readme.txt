Kaskus.co.id is the largest Indonesian Community. This plugin will show Hot Thread (Best Forum Post) automatically in Blog Sidebar.

== Description ==

Kaskus has a thousands of forum posting per day. The Kaskus team will choice 9 best forum post in a day. This plugin will show the best forum post (we called it as Hot Thread) in blog sidebar. 

Basically, this plugin call the rss from Yahoo Pipes. 

It safe, because it run under Yahoo server. The plugin will generate xml from this rss, and extract it into a widget. So you can load Kaskus Hot Thread automatically.

== Installation ==

1. Upload `kaskus-ht-widget` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Widget menu in Wordpress and put the Kaskus Hot Thread widget into your blog Sidebar.

== Frequently asked questions ==

= How it works? =
This plugin works with RSS system from Yahoo Pipes. Basicly, it read RSS feed of kaskus Hot Thread.
